# Installation

With `npm`:

```
npm install @rnacanvas/draw.strung
```

# Usage

All exports of this package can be accessed as named imports.

```javascript
// an example import
import { StrungElement } from '@rnacanvas/draw.strung';
```

## `class StrungElement`

A strung element
(e.g., a rectangle strung on a bond).
